**Spend Less Time on Hold, More Time on Health.**

Docuvoice is a web application that acts as an AI-powered medical voice assistant. It allows users to have natural voice conversations with various AI medical specialists to get instant health advice, symptom analysis, and treatment recommendations 24/7.

## ‚ú® Key Features

* **Natural Voice Consultations:** Talk to the AI just as you would a real doctor.
* **AI Medical Specialists:** Get specialized advice from AI personas, including:
    * General Physician
    * Pediatrician
    * Cardiologist
* **Detailed Medical Reports:** Receive a comprehensive, downloadable report after your consultation with:
    * Symptoms Discussed
    * Potential Diagnosis
    * Treatment Recommendations
* **AI Doctor Recommendations:** The system can suggest the right specialist for your symptoms.
* **24/7 Availability:** Get medical advice anytime, anywhere.

## üöÄ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* Node.js (v18 or later)
* npm, yarn, or pnpm

### Installation

1.  **Clone the repository:**
    ```sh
    git clone [https://github.com/Aryan7019/Docuvoice.git](https://github.com/Aryan7019/Docuvoice.git)
    cd Docuvoice
    ```

2.  **Install dependencies:**
    ```sh
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  **Set up environment variables:**
    Create a `.env.local` file in the root of your project and add the necessary API keys (e.g., for your AI provider).
    ```.env.local
    NEXT_PUBLIC_AI_API_KEY=YOUR_API_KEY_HERE
    ```

4.  **Run the development server:**
    ```sh
    npm run dev
    # or
    yarn dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
